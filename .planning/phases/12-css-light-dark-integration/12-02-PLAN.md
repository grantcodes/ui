**reset.css changes:**

The existing `:root` block (lines 223-229) already has `color-scheme: light dark;`. Add `.dark` and `.light` class overrides AFTER the `:root` block. Insert these rules after line 229 (the closing `}` of the `:root` block):

```css
/* Force dark mode on any element or subtree */
.dark {
	color-scheme: dark;
}

/* Force light mode on any element or subtree */
.light {
	color-scheme: light;
}
```

These declarations tell the browser to force-resolve `light-dark()` CSS values to the specified mode. When `.dark` is applied to an element, all `light-dark()` tokens within that subtree resolve to their dark value. Same for `.light`.
